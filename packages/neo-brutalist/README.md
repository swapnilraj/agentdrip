# @agentdrip/neo-brutalist

Brutalism's rebellious energy meets modern playfulness. Thick black borders, hard drop shadows, saturated background colors, and an intentionally "undesigned" feel.

## Usage

```js
import skill from '@agentdrip/neo-brutalist/skill';
import skin from '@agentdrip/neo-brutalist/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#FFFDF7` |
| fg      | `#1A1A1A` |
| accent  | `#FF6B9D` |

**Fonts:** Space Grotesk, DM Mono

**Tags:** bold, colorful, playful, borders, drop-shadow
