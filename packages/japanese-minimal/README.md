# @agentdrip/japanese-minimal

Ma -- the conscious use of negative space as a compositional element. Inspired by wabi-sabi and Japanese interior design.

## Usage

```js
import skill from '@agentdrip/japanese-minimal/skill';
import skin from '@agentdrip/japanese-minimal/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#F7F5F0` |
| fg      | `#2C2C2C` |
| accent  | `#B8564F` |
