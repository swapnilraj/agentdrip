# @agentdrip/editorial

The magazine spread brought to screen. Typography is expressive and varied. Layout is asymmetric but intentional.

## Usage

```js
import skill from '@agentdrip/editorial/skill';
import skin from '@agentdrip/editorial/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#FAFAF8` |
| fg      | `#1C1C1C` |
| accent  | `#D4380D` |

**Fonts:** Playfair Display, Source Sans 3, Source Serif 4

**Tags:** magazine, typographic, asymmetric, serif, print-inspired
