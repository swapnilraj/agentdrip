# @agentdrip/y2k-aesthetic

The internet as a frontier -- chaotic, glossy, and optimistic. Chrome gradients, neon glows, and early-2000s digital excess.

## Usage

```js
import skill from '@agentdrip/y2k-aesthetic/skill';
import skin from '@agentdrip/y2k-aesthetic/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#1A0A2E` |
| fg      | `#FFFFFF` |
| accent  | `#00CCFF` |

**Fonts:** Orbitron, VT323

**Tags:** neon, glossy, dark, retro, maximal
