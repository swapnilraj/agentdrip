# @agentdrip/dark-luxury

Premium, restrained, and atmospheric. Inspired by Linear, Vercel, and Stripe. The dark background is a deliberate stage for content to glow against.

## Usage

```js
import skill from '@agentdrip/dark-luxury/skill';
import skin from '@agentdrip/dark-luxury/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#09090B` |
| fg      | `#FAFAFA` |
| accent  | `#6366F1` |
