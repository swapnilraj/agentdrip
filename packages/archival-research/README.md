# @agentdrip/archival-research

The interface as a scholarly tool. Information density is a feature, not a bug. Inspired by Wikipedia, Are.na, and academic databases.

## Usage

```js
import skill from '@agentdrip/archival-research/skill';
import skin from '@agentdrip/archival-research/skin';
```
