# @agentdrip/retro-pixel

The 8-bit and 16-bit era brought to the web. Everything is built on a visible pixel grid. Pixels are celebrated, never anti-aliased.

## Usage

```js
import skill from '@agentdrip/retro-pixel/skill';
import skin from '@agentdrip/retro-pixel/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#2B2B3D` |
| fg      | `#E8E8D0` |
| accent  | `#6ABE30` |

**Fonts:** Press Start 2P, VT323

**Tags:** pixel, 8-bit, retro, dark, chunky
