# @agentdrip/organic-soft

Warmth, comfort, and approachability. Everything has rounded edges, soft shadows, and pastel gradients. The antithesis of sharp corporate design.

## Usage

```js
import skill from '@agentdrip/organic-soft/skill';
import skin from '@agentdrip/organic-soft/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#F8F5FF` |
| fg      | `#3D3455` |
| accent  | `#B8A9E8` |

**Fonts:** DM Sans, Nunito

**Tags:** pastel, rounded, soft, warm, friendly
