# @agentdrip/brutalist

Raw, honest, structural. The design exposes its own construction -- it looks like a blueprint or database dump.

## Usage

```js
import skill from '@agentdrip/brutalist/skill';
import skin from '@agentdrip/brutalist/skin';
```

## Design Tokens

| Token   | Value     |
|---------|-----------|
| bg      | `#FFFFFF` |
| fg      | `#000000` |
| accent  | `#0000FF` |

**Fonts:** Space Mono, Helvetica

**Tags:** monospace, raw, structural, high-contrast, uppercase
