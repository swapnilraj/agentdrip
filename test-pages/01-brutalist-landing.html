RAWDB.

An embedded database that does not hide its internals. No ORM. No migrations. No abstraction layers. You write queries. It returns data. 14kb.

Install

npm install rawdb
Features

01 / Zero Abstraction

No models. No schemas. No decorators. You put data in. You get data out. The API is six functions. Learn them in ten minutes.

02 / File-Based

Every collection is a file on disk. You can read it with cat. You can edit it with vim. You can version it with git. No server process.

03 / Typed Queries

Full TypeScript support. Query results are typed at compile time. No runtime validation overhead. The types are the documentation.

04 / Concurrent Reads

Lock-free reads. Write-ahead log for mutations. Handles thousands of concurrent readers without breaking a sweat. Single-writer guarantee.

05 / No Dependencies

Zero runtime dependencies. Ships as a single file. Works in Node, Deno, and Bun. The install is the entire product.

06 / Plaintext Format

Data stored as newline-delimited JSON. Human readable. Grep-friendly. No proprietary binary format. Your data is always yours.

Usage 
import { open, put, get, query, del, close } from 'rawdb';

const db = open('./data');
put(db, 'users', { id: '001', name: 'admin' });
const user = get(db, 'users', '001');
const all = query(db, 'users', u => u.name !== null);
close(db);
